# Tif Converter – Unreal Engine GeoTIFF Heightmap Plugin

Tif Converter converts GeoTIFF (.tif / .tiff) elevation data into 16-bit heightmaps ready for Unreal Landscapes.

The plugin normalizes elevation data, optionally applies smoothing, and automatically configures correct texture settings.

## Features

- Convert GeoTIFF to 16-bit PNG
- Automatic elevation normalization
- Optional smoothing
- Landscape-ready texture configuration
- Scale calculation assistance

## Installation

1. Copy the Tif_Converter folder into:
   <ProjectFolder>/Plugins/
2. Enable the plugin in Edit → Plugins
3. Restart Unreal Engine

## Quick Start

1. Open the Editor Utility Widget
2. Convert your GeoTIFF
3. Import the generated PNG in Landscape Mode
